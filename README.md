🛒 E-Commerce Sales Data Analysis

An end-to-end Data Analytics project using Python to analyze Indian E-Commerce sales data.
This project focuses on sales performance, customer behavior, revenue trends, and business insights through data cleaning, EDA, and visual dashboards.

📊 Project Highlights
📦 Analyzed 1,000 E-Commerce Orders
💰 Total Revenue: ₹5.94 Crore
📈 Created 8 Professional Visual Charts
🏷 Identified Top Categories & Cities
💳 Analyzed Payment Methods & Order Status
📊 Exported Business Reports to Excel
🛠 Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Excel
📁 Project Files
ecommerce_analysis.py → Main Python Script
ecommerce_sales_data.csv → Dataset
ecommerce_dashboard.png → Visual Dashboard

🚀 How to Run
pip install pandas numpy matplotlib seaborn openpyxl
python ecommerce_analysis.py
